### 概述

在Java中提供了四种访问权限，使用不同的访问权限修饰符修饰时，被修饰的内容会有不同的访问权限

+ public：公共的。
+ protected：受保护的
+ default：默认的
+ private：私有的

### 不同权限的访问能力

|              | public | protected | (default) | private |
| ------------ | :----: | :-------: | :-------: | :-----: |
| 同一个类     |   √    |     √     |     √     |    √    |
| 同一个包     |   √    |     √     |     √     |         |
| 不同包子类   |   √    |     √     |           |         |
| 不同包非子类 |   √    |           |           |         |

> (default)并不是关键字“default”，而是根本不写。

编写代码时，如果没有特殊的考虑，建议这样使用权限：

+ 成员变量使用private，隐藏细节。
+ 构造方法使用public，方便创建对象。
+ 成员方法使用public，方便调用方法。
